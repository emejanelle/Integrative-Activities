What is the difference between var, let, and const?

JavaScript Variables can be declared in four (4) ways: (a)Automatically, (b) Using var, (c) Using let, (d) Using const. These variables are used for storing data.

Var - Declares variables with function or global scope and allows re-declaration and updates within the same scope. Re-declaration of variable/s overwrites the previous value or data it stores.
Let - Declares variables with block scope, allowing updates but not re-declaration within the same block. Block-level scoping means the variable is only accessible within the block (like loops or conditionals) where it is declared.
Const - or constant, declares block-scoped variables that cannot be reassigned after their initial assignment.
