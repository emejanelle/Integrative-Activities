What is the role of the DOM in JavaScript?

HTML DOM or Document Object Model is a standard object model and programming interface used to manipulate and/or update HTML elements and attribute, as well as the CSS styles, through the JavaScript file.

How can JavaScript modify HTML content dynamically?

DOM provides methods and properties that allows the JS to modify HTML content dynamically.
